## 部署

代码推送到 GitHub 后，Vercel 和 Cloudflare Pages 都会自动部署，无需额外操作。

## 工作流程

1. 用户提需求 →
2. 修改本地源代码 →
3. `npm run build` 验证编译通过 →
4. `git push` 推送到 GitHub →
5. Vercel + Cloudflare Pages 自动部署
